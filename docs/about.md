## 项目定位

Mioku 是一个基于 `mioki` 的机器人框架，围绕 Agent 能力构建，在 `mioki` 的基础上规范了服务和AI的使用标准便于插件的分发，同时提供了常用的系统级服务例如屏幕截图和插件帮助，降低框架占用。

Mioku 取名自其依赖的基础包名 mioki，结合其初音未来主题，将"i"改为"u"，在寓意主题的同时代表开源(Open) ~~和某神人韩国网红没有关系~~

## 碎碎念

- `Mioku`作为一个轻量的、个人维护的框架，和别的框架例如`NoneBot`、`AstrBot`自然在功能完善度和社区贡献者数量上没得比，但是`Mioku`速度快、开发插件简单、插件容易分发、轻量...这些已经够了。
- 框架仓库根目录已经存有`AGENT.md`和`CLAUDE.md`，你可以使用任意一款 Vibe Coding 软件例如 Codex、Claude Code、OpenCode等等，选用一个聪明的模型，简单描述你希望的插件功能，AI就会帮助你实现，并且得益于`TypeScript`的类型定义，AI生成的插件可用率较高。
